# Kubernetes course
This project contains some exercises that are helpful to understand main concepts of K8.

## How to execute
Requirements are the following:
 - Docker 
 - Kubernetes 
 - Minukube (Cluster)

Each folder contains different cases, commands that are used during that execution are attached:

By filename 

- kubectl apply -f [filename]

Operations with PODS/DEPLOYMENT/RS

- kubectl get 
- kubectl describe 
- kubectl delete
- kubectl create
- kubectl edit 
- kubectl logs
